    css简介

    2022/6/8

    # 基础使用介绍

    css指层叠样式表，可控制html在网页中的表现，丰富网页内容。

    # 基础语法

    <!--选择器（指向控制的元素：此处修改<a></a>标签）-->
a {
  color: red; // 属性+值（如颜色：红色）
  background-color: blue; // 多个属性';'分隔
}

    简单说就是：由选择器选择不同元素，并对不同属性配置属性值，控制在页面的展示形式

    # 三种使用方式

    一、文件引入

    <link href="main.css">

    二、内部样式

    <style>
a {
  color: red;
}
</style>

    三、内联样式

    <a style="color:red">我是红色</a>

    # 注意点

    1. 覆盖：样式表从上往下加载，遇到相同 选择器+属性，后者覆盖前者
    2. 优先级：根据不同选择器组合，会有展示优先级，优先级高覆盖优先级低样式

    总结：css属性较多，本章仅简单介绍。css入门简单，但是写好却不容易。
